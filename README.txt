MicroORM
-----------

Small and Simple ORM for SQLite.



Installing MicroORM to your local copy of Python
------
1) Open a terminal and navigate to the downloaded MicroORM folder
2) Build a distribution file by running the following command: python3 setup.py sdist
3) Install the distribution by running the following command: sudo python3 setup.py install

In order to use MicroORM in your python project, you must include the line "import MicroORM"
